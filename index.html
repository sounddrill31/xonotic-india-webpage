WHATWE'RE BUILDING

We're trying to rebuild what we had: a decent Xonotic server hosted in India with low ping for Indian players. Nothing fancy, just a place where you can frag without lag.

The server's been running since 2023, survived admin burnout and technical issues. Maps might be outdated, some stuff's probably broken, but it's still there. We're working on fixing things up.

The goal is simple: get enough people playing so matches actually happen. Bridge chat to Discord/Telegram/IRC so the community can coordinate. Maybe run some tournaments if enough people show up.

It's not about building the next big thing. It's about having a server that works, in India, for people who want to play arena FPS without dealing with 200ms ping.
